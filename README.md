
# Task Taker (Task Manager)

Task Taker is a web application that helps you to manage your day to day tasks.

## Tech Stack

**Frontend:** Angular, Bootstrap, SCSS, HTML

**Backend:** NestJs, Typescript, MySQL

  
## Features

- User Authentication
- Filter Tasks

## Environment Variables

To run this project locally, you will need to run xampp server and create SQL database with name `nestjs`





  
## Run Locally

Clone the project

```bash
  git clone https://github.com/priyanshuguptaa/task-taker.git
```

Go to the project directory

```bash
  cd task-taker
```

Go to todo-api directory

```bash
  cd todo-api
```

Install dependencies

```bash
  npm install
  
```

Go to todo-front directory

```bash
  cd todo-front
```

Install dependencies

```bash
  npm install
  
```

Start the server from todo-api directory

```bash
  npm run start:dev
```

Launch frontend from todo-front directory

```bash
  ng serve
```
